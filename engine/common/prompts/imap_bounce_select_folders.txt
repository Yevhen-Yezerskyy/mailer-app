Вот список IMAP-папок.
Скажи JSON-ом, какие папки надо читать, чтобы проверить все входящие письма, где может оказаться ответ на нашу рассылку.

Вход: JSON с полем:
- `folders`: полный список реальных папок mailbox.

Правила:
- Верни только JSON без пояснений.
- Формат ответа строго: `{"folders":["..."]}`.
- В `folders` включай только имена, которые уже есть во входном списке `folders`.
- Не добавляй дубликаты.
- Не включай папку `SerenityMailer`.
- Если подходящих папок нет, верни `{"folders":[]}`.
