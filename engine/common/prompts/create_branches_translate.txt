Переведи список немецких названий бизнес-категорий на указанный язык интерфейса.

Вход:
- JSON-объект с полями:
  - `target_lang`
  - `target_lang_name`
  - `items`

Расшифровка входа:
- `target_lang` = код языка, на который нужно перевести категории.
- `target_lang_name` = название целевого языка.
- `items` = список объектов вида:
  - `id`
  - `branch_name_de`

Что такое эти категории:
- Это реальные названия бизнес-категорий из немецких онлайн-справочников:
  - Gelbe Seiten — https://www.gelbeseiten.de/
  - 11880 — https://www.11880.com/
- Эти названия используются в B2B-системе подбора компаний.
- `branch_name_de` всегда является немецким названием категории.

Задача:
- для каждого элемента перевести `branch_name_de` на язык `target_lang_name`;
- переводить кратко, нейтрально и по-деловому;
- сохранять смысл именно как названия бизнес-категории;
- не делать пояснений, описаний и развернутых фраз;
- если название уже фактически общеупотребимо и не требует перевода, допускается аккуратно оставить его близким к оригиналу.

Формат ответа:
- строго JSON-объект;
- поле `translations` — массив объектов;
- каждый объект должен содержать:
  - `id`
  - `branch_name`
- `id` должен быть тем же, что и во входе;
- для каждого входного элемента должен быть один выходной элемент;
- без пояснений вне JSON.

Ограничения:
- не меняй `id`;
- не пропускай элементы;
- не добавляй новые элементы;
- не возвращай markdown;
- не возвращай комментарии;
- не возвращай ничего кроме JSON.

Самопроверка:
- в ответе сохранены все входные `id`;
- `branch_name` переведен на `target_lang_name`;
- ответ содержит только JSON.

Ответ строго JSON:
{
  "translations": [
    {"id": 1, "branch_name": "..."}
  ]
}
