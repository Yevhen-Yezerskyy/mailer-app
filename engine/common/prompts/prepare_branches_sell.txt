Понятия:
Продукт — продукт или услуга, которая предлагается Продавцом к продаже Покупателю за деньги.
Продавец — компания, которая продаёт Продукт Покупателям.
Покупатель — компания, которая покупает Продукт.

Категория — категория / отрасль / бранч, структурная единица каталога https://www.gelbeseiten.de/, которая содержит список компаний, которые в рамках данной задачи рассматриваются как потенциальные Покупатели. Категория — подмножество потенциальных Покупателей из множества компаний каталога https://www.gelbeseiten.de/.
Сделка — Покупатель из Категории покупает Продукт и платит деньги Продавцу, а Продавец продаёт Продукт и получает деньги от Покупателя в B2B бизнес-контексте.

!Важно: Категория не означает наличие спроса на Продукт; спрос выводится только на основе Правил 1–2 и веб-контекста.

Входные данные:
- описание Продукта и Продавца (ниже)
- подсказка для оценки Категорий
- список Категорий

Задача:
- Для каждой Категории из списка оценить вероятность Сделки.

Рейтинг:
- rate — целое число от 1 до 100
- меньший rate означает более высокую вероятность Сделки
- больший rate означает более низкую вероятность Сделки
- намеренно используй весь диапазон рейтинга 1–100
- оценка производится относительно всех Категорий https://www.gelbeseiten.de/ (≈780 Категорий), а не ограниченного переданного списка

Правила рейтингования:

Правило 1. Категория получает рейтинг 1–40, только если верно хотя бы одно:
- Сделка осуществляется для собственных нужд Покупателя
- Сделка осуществляется для перепродажи как часть B2B-контекста
- Сделка осуществляется для перепродажи вследствие нехватки ресурсов Покупателя

Если ни одно из условий не выполнено, но есть сомнение → рейтинг 40–60
Если ни одно из условий не выполнено и сомнений нет → рейтинг 60–100

Правило 2. Категория может получить рейтинг 1–30, только если верно хотя бы одно:
- Сделка является срочной для собственных нужд Покупателя
- Сделка является срочной для нужд клиентов Покупателя (последующая перепродажа)
- Такие Сделки осуществляются Покупателем с другими продавцами регулярно (повторяющиеся / стандартные / рамочные) и являются частью бизнес-контекста Покупателя

Если ни одно из условий не выполнено, но есть сомнение → рейтинг 30–50
Если ни одно из условий не выполнено и сомнений нет → рейтинг 50–100

Правило 1 и Правило 2 являются независимыми ограничениями рейтинга и применяются одновременно.

При оценке учитывай все факторы, влияющие на вероятность Сделки:
- высший приоритет: типичность Сделок в Категории
- средний приоритет: способность и готовность Покупателей в Категории платить
- низкий приоритет: скорость принятия решения и заключения Сделки в Категории
- общий контекст каталога https://www.gelbeseiten.de/
- используй веб-поиск для оценки Продукта и Продавца
- используй веб-поиск для оценки Категорий, источник: https://www.gelbeseiten.de/

Инструкции:
- используй ТОЛЬКО id и name из переданного списка Категорий

Запреты:
- совпадение Категории, Покупателя или Категории и Продукта запрещено использовать как единственное основание для улучшения (уменьшения) рейтинга
- никаких новых Категорий не добавляй
- не выравнивай оценки и не избегай низких значений

Формат ответа (СТРОГО, только JSON, без текста):
