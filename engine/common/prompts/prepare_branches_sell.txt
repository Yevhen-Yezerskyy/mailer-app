Понятия:

Продукт — продукт или услуга, которая предлагается Продавцом к продаже Покупателю за деньги.

Продавец — сторона Сделки, которая продаёт Продукт Покупателям.

Покупатель — сторона Сделки, которая покупает Продукт у Продавца, для себя или дла последующей перепродажи.

Сделка — Покупатель из Категории покупает Продукт и платит деньги Продавцу, а Продавец продаёт Продукт и получает деньги от Покупателя.

Категория — это поисковая рубрика каталога https://www.gelbeseiten.de/, списки компаний доступны по "PLZ/Ort", 
в URL-шаблонах вида https://www.gelbeseiten.de/Branchen/{category}/{Ort} и https://www.gelbeseiten.de/Suche/{category}/{Ort}.
Название Категории обозначает, что компании предлагают рынку, их профиль продаж. Категория не определяет спрос на Продукт и не задаёт роль компаний как Покупателей.


Входные данные:
- описание Продукта и Продавца (ниже)
- подсказка для оценки Категорий
- список Категорий для оценки

Задача:
- Для каждой Категории из списка оценить вероятность Сделок относительно всех Категорий.

Рейтинг:
- rate — целое число от 1 до 100
- меньший rate означает более высокую вероятность Сделок
- больший rate означает более низкую вероятность Сделок
- намеренно используй весь диапазон рейтинга 1–100
- оценка производится относительно всех Категорий https://www.gelbeseiten.de/ (≈780 Категорий), а не ограниченного переданного списка
- Если Категория описывает прямых конкурентов Продавца, такие компании по умолчанию не являются Покупателями Продукта; для этой Категории запрещён rate < 50

Правила рейтингования:

Правило 1. Категория получает рейтинг 1–40, только если верно хотя бы одно:
- Сделка осуществляется для собственных нужд Покупателя
- Сделка осуществляется для перепродажи как часть B2B-контекста
- Сделка осуществляется для перепродажи вследствие нехватки ресурсов Покупателя

Если ни одно из условий не выполнено, но есть сомнение → рейтинг 40–60
Если ни одно из условий не выполнено и сомнений нет → рейтинг 60–100

Правило 2. Категория может получить рейтинг 1–30, только если верно хотя бы одно:
- Сделка является срочной для собственных нужд Покупателя
- Сделка является срочной для нужд клиентов Покупателя (последующая перепродажа)
- Такие Сделки осуществляются Покупателем с другими продавцами регулярно (повторяющиеся / стандартные / рамочные) и являются частью бизнес-контекста Покупателя

Если ни одно из условий не выполнено, но есть сомнение → рейтинг 30–50
Если ни одно из условий не выполнено и сомнений нет → рейтинг 50–100

Правило 1 и Правило 2 являются независимыми ограничениями рейтинга и применяются одновременно.

Инструкции:
- используй ТОЛЬКО id и name из переданного списка Категорий

Формат ответа (СТРОГО, только JSON, без текста):

[
  {"id": 123, "name": "Example_name_a", "rate": 7},
  {"id": 456, "name": "Example_name_b", "rate": 12}
]
