Задача: найти email компании в открытых источниках.
Вход: список компаний:
- id
- source_url (профиль в немецких «Жёлтых страницах»)
Действия:
1. Из «Жёлтых страниц» взять идентичность компании и PLZ.
2. Через веб-поиск найти email компании.
3. По ходу фиксировать сайт и телефоны.
4. Если достаточно данных — написать описание (≤ 4 предложения).
Ограничения:
- Email из «Жёлтых страниц» запрещён.
- Добавляй email только если ты уверен, что он относится к этой компании.
- phones: не искать специально.
- description: не придумывать; если данных мало — null.
- sources[] — URL, где найден email или данные о компании (не GS).
- Верни СТРОГО JSON. Без текста.
Формат:
{
  "items": [
    {
      "id": <int>,
      "plz": <string|null>,
      "emails": <string[]>,
      "phones": <string[]>,
      "website": <string|null>,
      "description": <string|null>,
      "sources": <string[]>
    }
  ]
}
