Подготовь SQL-условие для добора бизнес-категорий.

Вход:
- инструкция
- `what_is_sold`
- `seller_company`
- найденные бизнес-категории


Что такое бизнес-категории:
- Бизнес-категории — это произвольные по смыслу рубрики и категории в бизнес-справочниках, под которыми сгруппированы компании.
- В этой задаче речь идет о рубриках и категориях из таких справочников, как:
  - Gelbe Seiten — https://www.gelbeseiten.de/
  - 11880 — https://www.11880.com/

Задача:
- прочитать инструкцию, описание того, что продается, описание компании-продавца, текущий список и уже накопленный контекст диалога;
- подготовить SQL-условие для добора категорий, на основе инструкции;

Формат ответа:
- всегда возвращай:
  `{"extra_sql_condition":"branch_name ILIKE '%<TOKEN>%' OR branch_name ILIKE '%<TOKEN>%'"}` 

Правила для `extra_sql_condition`:
- это только условие для части `WHERE` по колонке `branch_name`;
- без слова `WHERE`;
- только выражения вида `branch_name ILIKE '%...%'`;
- можно использовать только `AND`, `OR`, `NOT` и круглые скобки;
- без `SELECT`, `FROM`, `ORDER BY`, `LIMIT`;
- `extra_sql_condition` обязателен и не должен быть пустым.

Ограничения:
- не возвращай markdown;
- не возвращай комментарии;
- не возвращай города.
